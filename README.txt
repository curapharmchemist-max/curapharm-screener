CuraPharm Pharmacy First hosted app package

Upload all of these files to the root of your GitHub repository:
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

GitHub Pages settings:
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

On iPad:
- Open the site in Safari
- Tap Share
- Tap Add to Home Screen
- Launch from the home screen icon

This package:
- opens full-screen when launched from home screen
- uses CuraPharm colours
- caches core files for offline use after first load
